# 인간은 색을 어떻게 인지할까?
**간상세포**와 **원추세포**가 각각 **명암**과 **색**을 감지하여 색을 인지한다.

이렇게 색을 인지하는 능력에 이상이 있는 상태를 **색각이상**이라고 한다.

![Source:BenQ Korea](https://image.benq.com/is/image/benqco/01-what-is-color-1?$ResponsivePreset$&fmt=png-alpha)
# 색각 이상의 원인
- **선천적 색각이상**: 유전적으로 원추세포 기능 저하 또는 손상으로 인한 색각이상
- **후천적 색각이상**: 후천적 요인으로 인한 당뇨, 녹내장 또는 뇌 손상으로 인한 색각이상
# 색각이상의 분류
## 원추세포 이상 정도에 따른 분류
- **이색형 색각자(Dichromacy);** 세 가지 원추세포 중 한 가지가 없는 경우
- **이상 삼색형 색각자(Anomalous Trichromacy);** 세 가지 원추세포가 모두 존재하지만 한 가지가 비정상적인 경우
- **단색형 색각자(Monochromat);** 한 가지 원추세포만 가지고 있거나 원추세포가 전혀 없는 경우
## 원추세포 이상 유형에 따른 분류
- **제1 이상**
    - 적색약(protanomaly): 적색 원뿔세포의 기능이 약화된 경우
    - 적색맹(protanopia): 적색 원뿔세포의 기능이 완전히 손상된 경우
- **제2 이상**
    - 녹색약(deuteranomaly): 녹색 원뿔세포의 기능이 약화된 경우
    - 녹색약(deuteranomaly): 녹색 원뿔세포의 기능이 약화된 경우
- **제3 이상**
    - 청색약(tritanomaly): 청색 원뿔세포의 기능이 약화된 경우
    - 청색맹(tritanopia): 청색 원뿔세포의 기능이 완전히 손상된 경우
- **전색맹(achromatopsia);** 모든 색을 식별할 수 없는 경우
## 원인에 따른 분류
- **선천적, 후천적 색각이상**
- **대뇌 색맹(Cerebral achromatopsia);** 눈의 망막 이상이 아닌, 뇌의 시각 피질손상으로 인해 발생하는 후천적 색각이상

## Theme1. 세포 리프로그래밍을 통한 치료_*김채혁*
    "기존 세포를 다른 세포로 바꾸는 방법을 통해 색각 이상을 치료해보자!"
### 개념 및 용어 설명
세포 리프로그래밍 방법과 도구를 설명하는데 사용된 용어들을 알기 쉽게 설명하였습니다.

**1. 세포 -> 만능 줄기세포 -> 다른 세포 방법** 
- **야마나카 인자**: 세포를 **만능 줄기세포**로 바꿈. 아래 있는 4개의 유전자를 넣으면 세포가 **'아무 세포로나 될 수 있는 상태'** 가 됨.
    - **Oct-4**: 세포한테 아직 아무것도 하지 마라고 알려주는 유전자
    - **Sox2**: 세포의 상태를 유지하게 도와주는 유전자
    - **Klf4**: 증식, 분화 사멸과 같은 세포의 여러 성질을 안정화 시켜주는 유전자
    - **c-Myc**: 세포를 빠르게 증식하게 하는 유전자

**2. 세포 -> 다른세포 방법**
- **직접 교차분화**: 세포를 **만능 줄기세포 단계 없이** 바로 다른 종류의 세포로 바꾸는 방법. 아래는 어떤 세포를 바로 신경세포로 변화시킬 때 쓰이는 유전자들이다.
    - **Ascl1**: 세포를 변하게 하는 유전자.
    - **Brn2, Myt1l**: 신경세포의 정체성을 강화하고 성숙을 촉진하는 역할을 하는 유전자.

**3. 세포의 진로 안내 표지판(세포한테 "어디로 자라라"라고 말해줌)**
- **BMP**: **표피, 중배엽, 비신경 외배엽**으로 자라라고 유도하는 신호 물질
- **TGF-β**: **몸 안쪽 장기계열(중배엽, 내배엽)** 으로 자라라고 유도하는 신호 물질
- **WNT**: 세기에 따라 세기가 높을 수록 각각 **후뇌와 척수, 전뇌와 눈, 망막**으로 자라라고 유도하는 신호 물질
## Theme2. 유전자를 이용한 치료_*정시원*
    "특수한 유전자를 활용하여 색각 이상을 직접 치료해보자!"
### 개념 및 용어 설명
- **아데노 부속 바이러스(VAA)**: 안전성이 높아서 유전자를 효율적을 옮겨질 수 있는 바이러스
- **유전자 전달체**: 치료형 유전자를 안전하고 효율적으로 보내 줌.
    - 바이러스 성 벡터
    - 비바이러스 성 벡터
- **옵신 단백질**: 빛을 감지하는 광수용체 단백질. 시각세포에서 빛과 반응하여 전기 신호로 바꾸는 역할을 함.

## Theme3. 색상 필터를 이용한 반도체 렌즈_*왕예린*
    "특수한 렌즈를 활용하여 색을 보정해보자!"
### 개념 및 용어 설명
- **크로마젠 렌즈(색약 보정 렌즈)**: 렌즈 가운데에 있는 빨간색 필터가 특정 파장의 색을 부분할 수 있도록 도움. 현재 시중에서 많이 볼 수 있음.
- **메타렌즈**: 굴곡이 없는 평면렌즈. 빛의 파장보다 작은 나노 입자(비금속 유전체 소재, Nanord, Nanopiller 등)를 표면에 배열하여 빛을 특정한 지점으로 모아 줌.
    - **메타 원자**: 자연계 물질의 기하학적 특성을 활용하여 인위적으로 특정 물성을 지니도록 설계된 구조체
    - **메타 표면**: 메타원자가 일정한 주기로 배열된 표면. 입사된 빛의 방향을 특정 지점으로 모아주는 특성을 가짐.
- **나노 포토닉 구조**: 광학적 성질을 가질 수 있는 물질을 나노 미터 스케일로 만든 인공 구조물. 광학적 현상이 발생하고 구조색이 나타남
    - **구조색**: 빛의 회절이나 간섭과 같이 순전히 물리적인 원리에 의해 나타나는 색
    - **나노 포토닉 구조색 필터**: 2개의 적층된 나노 포토닉 구조로 만듦. 색상의 미세 조정과 고순도의 색상 표현이 가능한 필터
- **하이드로겔 렌즈**: 하이드로겔의 장점에 높은 산소 투과율을 더해 장시간 착용에도 눈 건강에 유리한 최신 렌즈
    - **하이드로겔**: 물을 다량 함유하여 높은 수분감과 유연성을 가지며, 생체 적합성이 뛰어난 3차원 망상 구조의 친수성 고분자 물질

## Theme5. Sonification을 이용한 색각 이상자를 위한 보조장치 제작_*김주원*
    '그림을 귀로 들을 수 있을까?' 
현대 의학의 기술로는 영원히 **색인지**가 불가능한 색각 이상자들을 위해 **감각의 전이(시각을 청각으로)** 를 통해 색인지를 돕고 싶었습니다.  

**그림을 소리로 바꿔주는 장치**를 직접 구현해봤습니다.
![main.py 실행 화면](https://soongilwiki.com/image/7cf269fefd90a7fa705da85b364519f438e5f0fa8ebe5f00130f8912.png)~~이런 거 넣으면 뭔가 있어보인다.~~

### 개념 및 용어 설명
    보조장치를 실제로 구현하려다보니 다소 전문적이고 기술적인
    부분들이 내용에 추가되었습니다. 
- **Sonification(음향화)**\
    소니피케이션이란, 정보 전달이나 데이터를 지각하게 하는데 음향을 이용하는 겁니다. 그러니까 어떤 데이터를 **청각을 통해 지각**할 수 있도록 하는 거죠.
- **자연어**\
    자연어란, 일상에서 우리가 사용하는 모든 언어를 말합니다. 말과 텍스트는 물론 이미지 또한 자연어에 해당합니다. 이런 자연어를 컴퓨터가 알아먹기 위해선 일종의 번역 과정이 필요한데, 이를 **자연어 처리**라고 합니다.
- **코사인 유사도(Cosine Similarity)**\
    코사인 유사도란, 두 벡터 사이의 각도를 바탕으로 두 벡터가 얼마나 유사한 지 판단하는 기법입니다. 코사인 유사도는 **스칼라 곱(내적, Dot Product)** 을 이용하기에 이를 통해 나온 값은 -1과 1사이의 스칼라 값을 갖게 됩니다. '이미지'와 '감정'의 벡터를 행렬로 표현한 다음 코사인 유사도를 통해 '감정 강도'를 추출할 수 있습니다. 
- **벡터와 스칼라**\
    벡터는 방향과 크기를 갖는 물리량으로 변위, 속도 등이 있으며 스칼라는 크기만 갖는 물리량으로 이동거리, 온도 등이 있습니다.
- **임베딩(Embedding)**\
    임베딩이란, 어떤 데이터를 기계가 보다 쉽게 이해할 수 있도록 **더 낮은 차원의 벡터로 변환**시키는 기술 또는 **변환시킨 그 벡터**를 의미합니다. 자연어 처리에서 매우 핵심적인 기술입니다.
- **인코더(Encoder)**\
    인코더란, 입력 데이터를 컴퓨터 내부에서 사용할 수 있는 코드로 변환하는 장치를 말합니다. 반의어로 디코더(Decoder)가 있습니다.
- **매핑(mapping)**\
    매핑이란, 서로 다른 두개의 요소나 데이터를 **서로 대응**시키는 것을 말합니다. 1대 1로 대응될 수도 있고, 1대 다로 대응될 수도 있습니다.
- **매개변수**\
    매개변수란, **서로 다른 두 변수를 묶어주는 변수**를 말합니다. 이 모델에선 'psych(ology)'가 감정과 선택할 파일을 묶어주는 매개변수의 역할을 수행합니다. (보통 프로그래밍에선 정의하는 모든 변수를 매개변수라 하지만 여기선 그런 의미는 아닙니다.)
- **HSV(Hue Saturation Value)**\
    HSV란, **Hue(색상) Saturation(채도) Value(명도)** 의 두문자어로 색을 나타내는 방식 중 하나입니다. RGB와 같은 레벨의 개념입니다.
- **클러스터링과 클러스터(Clustering, Cluster)**\
    Clustering(군집화)란, 말 그대로 **유사한 데이터들을 하나라 묶는 기술**입니다. 이 모델에선 K-Means 클러스터링 방식을 사용했습니다. 이렇게 만들어진 데이터 집단을 Cluster(군집)라 부릅니다.
- **인터렉션(Interaction)**\
    인터렉션이란, 직역하면 '상호작용'으로 디지털 환경에선 사용자와 상호작용하는 과정이나 그렇게 만들어진 제품을 의미합니다.
#### 임베딩 과정과 합성 곱 신경망을 이용한 문자 임베딩
    CLIP과 Chat-GPT를 비롯한 인공지능의 동장 방식에 대해 좀 더 자세히 알고 싶은 사람만 보세요! 정신 건강에 해롭습니다!
- **인코딩(Encoderding)**\
    인코딩이란, 입력 데이터를 컴퓨터 내부에서 사용할 수 있는 코드로 변환하는 과정을 말합니니다. 인간이 사용하는 언어를 컴퓨터가 사용할 수 있도록 바꾸는 과정 등을 예시로 들 수 있겠습니다.
이러한 인코딩을 수행하는 장치를 인코더(Encorder)라고 하고, 이와 반대 과정을 수행하는 작업은 디코딩(Decoderding)이라고 합니다.

이 중에서도 특별히 어떤 데이터를 벡터화 하는 것 또는 그렇게 만들어진 벡터를 **임베딩**이라고 하는데, 오늘날 우리에게 친숙한 거의 대부분의 LLM(Large Language Model, 챗지피티, 제미나이 등..)에서도 사용되는 **합성 곱 신경망을 이용한 문자 임베딩** 과정에 대해 살펴봅시다.

(이렇게 인공 신경망을 활용한 인공지능을 **딥러닝**이라고 합니다.)
