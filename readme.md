# 인간은 색을 어떻게 인지할까?
**간상세포**와 **원추세포**가 각각 **명암**과 **색**을 감지하여 색을 인지한다.

이렇게 색을 인지하는 능력에 이상이 있는 상태를 **색각이상**이라고 한다.

![Source:BenQ Korea](https://image.benq.com/is/image/benqco/01-what-is-color-1?$ResponsivePreset$&fmt=png-alpha)
# 색각이상의 분류
## 원추세포 이상 정도에 따른 분류
- **이색형 색각자(Dichromacy);** 세 가지 원추세포 중 한 가지가 없는 경우
- **이상 삼색형 색각자(Anomalous Trichromacy);** 세 가지 원추세포가 모두 존재하지만 한 가지가 비정상적인 경우
- **단색형 색각자(Monochromat);** 한 가지 원추세포만 가지고 있거나 원추세포가 전혀 없는 경우
## 원추세포 이상 유형에 따른 분류
- **제1 이상**
    - 적색약(protanomaly): 적색 원뿔세포의 기능이 약화된 경우
    - 적색맹(protanopia): 적색 원뿔세포의 기능이 완전히 손상된 경우
- **제2 이상**
    - 녹색약(deuteranomaly): 녹색 원뿔세포의 기능이 약화된 경우
    - 녹색약(deuteranomaly): 녹색 원뿔세포의 기능이 약화된 경우
- **제3 이상**
    - 청색약(tritanomaly): 청색 원뿔세포의 기능이 약화된 경우
    - 청색맹(tritanopia): 청색 원뿔세포의 기능이 완전히 손상된 경우
- **전색맹(achromatopsia);** 모든 색을 식별할 수 없는 경우
## 원인에 따른 분류
- **선천적, 후천적 색각이상**
- **대뇌 색맹(Cerebral achromatopsia);** 눈의 망막 이상이 아닌, 뇌의 시각 피질손상으로 인해 발생하는 후천적 색각이상
# 색각 이상은 왜 발생할까?
- **선천적 색각이상**: 유전적으로 원추세포 기능 저하 또는 손상
- **후천적 색각이상**: 후천적 요인으로 인한 당뇨, 녹내장 또는 뇌 손상으로 인한 색각이상

## Sonification을 이용한 색각 이상자를 위한 보조장치 제작_*김주원*
    '그림을 귀로 들을 수 있을까?' 
현대 의학의 기술로는 영원히 **색인지**가 불가능한 색각 이상자들을 위해 **감각의 전이(시각을 청각으로)** 를 통해 색인지를 돕고 싶었다.  

그림을 소리로 바꿔주는 장치를 직접 구현해본다.
### 용어 설명
- **Sonification(음향화)**\
    소니피케이션이란, 정보 전달이나 데이터를 지각하게 하는데 음향을 이용하는 겁니다. 그러니까 어떤 데이터를 **청각을 통해 지각**할 수 있도록 하는 거죠.
- **자연어**\
    자연어란, 일상에서 우리가 사용하는 모든 언어를 말합니다. 말과 텍스트는 물론 이미지 또한 자연어에 해당합니다. 이런 자연어를 컴퓨터가 알아먹기 위해선 일종의 번역 과정이 필요한데, 이를 **자연어 처리**라고 합니다.
- **코사인 유사도(Cosine Similarity)**\
    코사인 유사도란, 두 벡터 사이의 각도를 바탕으로 두 벡터가 얼마나 유사한 지 판단하는 기법입니다. 코사인 유사도는 **스칼라 곱(내적, Dot Product)** 을 이용하기에 이를 통해 나온 값은 -1과 1사이의 스칼라 값을 갖게 됩니다. '이미지'와 '감정'의 벡터를 행렬로 표현한 다음 코사인 유사도를 통해 '감정 강도'를 추출할 수 있습니다. 
- **벡터와 스칼라**\
    벡터는 방향과 크기를 갖는 물리량으로 변위, 속도 등이 있으며 스칼라는 크기만 갖는 물리량으로 이동거리, 온도 등이 있습니다.
- **임베딩(Embedding)**\
    임베딩이란, 어떤 데이터를 기계가 보다 쉽게 이해할 수 있도록 **더 낮은 차원의 벡터로 변환**시키는 기술 또는 변환시킨 그 벡터를 의미합니다. 자연어 처리에서 매우 핵심적인 기술입니다.
- **인코더(Encoder)**\
    인코더란, 입력 데이터를 컴퓨터 내부에서 사용할 수 있는 코드로 변환하는 장치를 말합니다. 반의어로 디코더(Decoder)가 있습니다.
- **매핑(mapping)**\
    매핑이란, 서로 다른 두개의 요소나 데이터를 **서로 대응**시키는 것을 말합니다. 1대 1로 대응될 수도 있고, 1대 다로 대응될 수도 있습니다.
- **매개변수**\
    매개변수란, **서로 다른 두 변수를 묶어주는 변수**를 말합니다. 이 모델에선 'psych(ology)'가 감정과 선택할 파일을 묶어주는 매개변수의 역할을 수행합니다. (보통 프로그래밍에선 정의하는 모든 변수를 매개변수라 하지만 여기선 그런 의미는 아닙니다.)
- **HSV(Hue Saturation Value)**\
    HSV란, **Hue(색상) Saturation(채도) Value(명도)** 의 두문자어로 색을 나타내는 방식 중 하나입니다. RGB와 같은 레벨의 개념입니다.
- **클러스터링과 클러스터(Clustering, Cluster)**\
    Clustering(군집화)란, 말 그대로 **유사한 데이터들을 하나라 묶는 기술**입니다. 이 모델에선 K-Means 클러스터링 방식을 사용했습니다. 이렇게 만들어진 데이터 집단을 Cluster(군집)라 부릅니다.
- **인터렉션(Interaction)**\
    인터렉션이란, 직역하면 '상호작용'으로 디지털 환경에선 사용자와 상호작용하는 과정이나 그렇게 만들어진 제품을 의미합니다.
